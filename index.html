How It Works

This web app helps you get real-time information about TTC stops.

  1. Search for a stop by its name, Route or direction or browse nearby stops.
  2. Click on a stop to send a pre-formatted message to the TTC's SMS service.
  3. You'll receive a text message with the next arrival times for that stop.

It's that simple!
